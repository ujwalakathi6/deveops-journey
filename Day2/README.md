#### 🐧 Linux Commands

  * `ls`, `cd`, `pwd`
  * `mkdir`, `rm`
  * `cp`, `mv`
* Understood how to work in the terminal efficiently

---

#### 🌐 Nginx

* Learned about **Nginx** and its role as a:

  * Web server
  * Reverse proxy
  * Load balancer (basic idea)

---

#### ⚙️ Installing Nginx

* Installed Nginx on my system
* Verified installation using:

  ```bash
  nginx -v
  ```
* Started the server:

  ```bash
  sudo systemctl start nginx
  ```
* Checked in browser using:

  ```
  http://localhost
  ```

---

#### 🌍 Hosting a Webpage using Nginx

* Cloned a webpage from GitHub:

  ```bash
  git clone <repository-link>
  ```
* Moved project files to Nginx root directory:

  ```bash
  /var/www/html/
  ```
* Successfully hosted and viewed my own webpage locally 🎉
